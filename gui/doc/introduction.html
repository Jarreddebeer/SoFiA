Introduction

SoFiA, the Source Finding Application, is a graphical user interface for the WALLABY/DINGO HI source finding pipeline. Its main purpose is to provide a comfortable and easy-to-use way of creating and modifying parameter files for the pipeline. Once a parameter file has been loaded or created, the pipeline can directly be invoked from within the user interface. Alternatively, parameter files created with SoFiA can be saved to disk, and the pipeline can then be run from the command line.
