User Interface

Tool Bar

For convenience, SoFiA comes with a tool bar that allows the user to access frequently used actions with just a single mouse click. The tool bar can be hidden by unchecking the  Show Toolbar  option available either in the  Settings  menu or by right-clicking on the tool bar. The following actions are available from the tool bar:

  • New File
  • Open File
  • Save File
  • Run Pipeline
  • Abort Pipeline
  • Clear Messages
